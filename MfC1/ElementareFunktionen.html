Definition

Eine reelle Funktion \(f\) ist eine Zuordnungsvorschrift, die einer reellen Zahl \(x\) aus einer gegebenen Teilmenge \mathcal{D} eine reelle Zahl \(f(x)\) zuweist, notiert als f:\mathcal{D}\to\mathbb{R}\;,\;x\mapsto f(x) Dabei heißt \mathcal{D} der Definitionsbereich von \(f\). Die Zahl \(f(x)\) heißt Bild von \(x\) unter \(f\)

Die Zeile f:\mathcal{D}\to\mathbb{R}\;,\;x\mapsto f(x) wird gesprochen als "f von D nach R, x wird abgebildet auf f von x".

Häufig ist der Definitionsbereich reeller Funktionen gleich den reellen Zahlen, es werden also alle reellen Zahlen abgebildet. Es kann jedoch vorkommen, dass bei der Berechnung des Bildes von \(x\) durch Null dividiert werden muss, was nicht erklärt ist. Solche \(x\) dürfen nicht im Definitionsbereich liegen, weswegen reelle Funktionen im Allgemeinen nur auf einer Teilmenge \mathcal{D} der reellen Zahlen erklärt werden. Im einfachsten Fall ist \mathcal{D}=\mathbb{R}.

Ist der Definitionsbereich klar, so wird statt des vollen f:\mathcal{D}\to\mathbb{R}\;,\;x\mapsto f(x) nur die Zuordnungsvorschrift f(x) geschrieben.

Die Exponentialfunktion

Die Exponentialfunktion zur Basis \(a\) ist eine Funktion, die auf ganz \mathbb{R} erklärt ist. Sie bildet eine Zahl \(x\) ab auf eine Zahl \(a^x\), derart dass gilt \begin{eqnarray*} a^{x}\cdot a^{y} &=& a^{x+y}\\ a^1 &=& a \end{eqnarray*} Die Zahl \(1\) wird also auf die Zahl \(a\) abgebildet, alle anderen Werte ergeben sich aus der Rechenregel a^{x}a^{y}=a^{x+y}.

Beispiel

Sei \(a=2\). Für natürlichen Zahlen \(n\) ist die Exponentialfunktion zur Basis \(2\) die übliche \(n-\)te Potenz von \(2\), also \(2^1=2, 2^2=4, 2^3=8, 2^4=16\) und so weiter.
