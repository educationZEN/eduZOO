Integralsatz von Stokes

1. Sei F\subset\mathbb{R}^3 eine Fläche mit stückweise glatter Randkurve \partial F und \vec{v}:\mathbb{R}^3\to\mathbb{R}^3 ein Vektorfeld. (Die Konvention für die Orientierung von Randkurven vorausgesetzt, ) gilt: \iint\limits_F\mbox{rot}\ \vec{v}\cdot d\vec{O}= \int\limits_{\partial F}\vec{v}\cdot d\vec{s}

immer (die Gleichheit heißt Satz von Stokes) falls \vec{v} auf einer offenen Umgebung von F stetig ist. falls \vec{v} auf einer offenen Umgebung von F stetig differenzierbar ist.

2. Die Konvention für Orientierung von Randflächen ist:

Bei einer Fläche im \mathbb{R}^2 liegt das Gebiet rechts, wenn man die Randkurve im positiven Sinn durchläuft. Bei einer Fläche im \mathbb{R}^2 liegt das Gebiet links, wenn man die Randkurve im positiven Sinn durchläuft. Bei einer Fläche im \mathbb{R}^3 mit vorgegebenem Normalenvektor liegt bei positivem Umlaufsinn des Randes die Fläche links, wenn man von der Seite guckt, in die der Normalenvektor zeigt.
