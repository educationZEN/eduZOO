Mehrdimensionale Integration

1. Sei B\subset\mathbb{R}^2, \phi:B\to\mathbb{R}^3 eine Funktion.

\phi ist eine Parametrisierung der Fläche \phi(B). \phi(B) ist eine Fläche falls \phi(x,y)=(x,y,x+y) und B=[0,1]\times[0,1]. \phi(B) ist eine Fläche falls \phi(x,y)=(x,0,0) und B=[0,1]\times[0,1].

2. Sei B\subset\mathbb{R}^2 kompakt, \phi:B\to\mathbb{R}^3 eine Parametrisierung der glatten Fläche \phi(B)=:F und f:F\to\mathbb{R} eine Funktion, die die Massenverteilung von F beschreibt.

Die Oberfläche von F ist \iint_B\|\frac{\partial\phi}{\partial u}\times \frac{\partial\phi}{\partial v}\|du\ dv. Die Gesamtmasse von F ist \iint_B f(u,v)\|\frac{\partial\phi}{\partial u}\times \frac{\partial\phi}{\partial v}\|du\ dv. Die Gesamtmasse von F ist \iint_B f(\phi(u,v))\|\frac{\partial\phi}{\partial u}\times \frac{\partial\phi}{\partial v}\|du\ dv.
